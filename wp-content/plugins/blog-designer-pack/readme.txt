= This Blog Designer Pack contains 7 shorcode =

* <code>[bdp_post]</code> : Display blog post in grid view.
* <code>[bdp_post_list]</code> : Display blog post in list view.
* <code>[bdp_masonry]</code> : Display blog post in masonry layouts.
* <code>[bdp_post_slider]</code> : Display recent blog post in slider view.
* <code>[bdp_post_carousel]</code> : Display recent blog post in carousel view.
* <code>[bdp_post_gridbox]</code> : Display blog post in grid box layout.
* <code>[bdp_ticker]</code> : Display blog post in ticker.

= Common Shortcode Paramaters for all =

* **Limit:**
limit="10" ( ie Display 5 posts on your website and then paginations. To show all posts use limit="-1" )

* **Display by Category**
category="category_ID"] ( ie Display posts by their category ID )

* **Display category name:**
[logoshowcase cat_name="category name"] ( Display category name)

* **Design:**
design="design-1" (Select design for post design. There are 2 designs. design-1 and design-2 )

* **Show Blog Author:**
show_author="false" (Show post author or not. values are true or false)

* **Show Blog Category:**
show_category="false" (Show post category or not. values are true or false)

* **Show Blog Date:**
show_date="false" (Show post date or not. values are true or false)

* **Show Blog Content:**
show_content="false" (Show post content or not. values are true or false)

* **Content Words Limit:**
content_words_limit="40" (Set content words limit for post.)

* **Media Size:**
media_size="large" (Set media size for post. values are thumbnail, medium, large, full)

* **Show Read More:**
show_read_more="true" (Show read more button of post.  values are true or false)

* **Post Order:**
order="DESC" (Designates the ascending or descending order of the 'orderby' parameter. Defaults to 'DESC'. Values are "DESC" and "ASC") 

* **Post Order by :**
orderby="post_date" (Sort retrieved posts by parameter. Defaults to 'date (post_date)'. One or more options can be passed. 'none',ID','author','title','name',rand',date')

== Installation ==

1. Upload the 'blog-designer-pack' folder to the '/wp-content/plugins/' directory.
2. Activate the Blog Designer Pack plugin through the 'Plugins' menu in WordPress.
3. Add and manage blog post items on your site by clicking on the  'Post' tab that appears in your admin menu.
